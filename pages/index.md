### Welcome to Eris Nomic

This is a game of [Nomic](https://legacy.earlham.edu/~peters/writing/nomic.htm), a game of self-amendment.

The game is run online over Discord -- this website hosts the rules, which, given the nature of the game, are subject to change.

The rules of the game can be found [here](./rules/rules.html).
